# Synapsin

**Developer Tools That Actually Work**

We build practical tools that solve real developer problems. Sometimes offline, sometimes using APIs - whatever works best.

## 🚀 Our Tools

### [CodeSnip Vault](https://github.com/synapsin-ai/codesnip-vault)
Android app for managing code snippets on your phone. Because sometimes you need your code library when you only have your phone.
- 📱 Works offline completely
- 🏃 Execute JavaScript/HTML/CSS on device
- 📤 Export to JSON, CSV, PDF, Markdown
- 🎯 **Status**: Beta testing - [Join waitlist](mailto:hello@synapsin.ai)

### [Workspace Embeddings](https://github.com/mr-matthew-pohl/workspace-embeddings)
Semantic code search that actually understands what you're looking for.
- 🔍 Search by meaning: "authentication logic" finds auth code
- ⚡ Find performance issues: O(n²) loops, N+1 queries
- 🔒 Security scanner: SQL injection, XSS, hardcoded secrets
- 🏗️ Architecture detection: MVC, microservices, monoliths
- 🎯 **Status**: Open source, production-ready - [Try it now](https://github.com/mr-matthew-pohl/workspace-embeddings)

### Claude-Gemini Bridge
Let Claude Code use Gemini's massive context when needed. Right tool for the right job.
- 🧠 Claude for coding, Gemini for analysis
- 📊 Handle 1M+ token codebases
- 🔄 Automatic task routing
- 🎯 **Status**: Private beta - [Request access](mailto:hello@synapsin.ai)
