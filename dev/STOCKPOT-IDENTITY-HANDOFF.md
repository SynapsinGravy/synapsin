# Identity Handoff: obsidian-memory → **Stockpot** (public name)

2026-06-12. For the memory-system project (GitHub `mr-matthew-pohl/obsidian-memory`, no local folder here — drop this file in that repo). Full research data: `~/projects/NAMING-AND-BLURBS.md`.

## The rename is mandatory, not cosmetic

"obsidian-memory" cannot appear publicly. "Obsidian" is Dynalist Inc.'s trademark for the Obsidian note app, and this project is *literally an Obsidian-compatible markdown store* per its own README — a public "obsidian-memory" reads as an official Obsidian product: textbook likelihood-of-confusion (Microsoft's Obsidian Entertainment holds software marks too). The private repo name is fine; the public identity is not. Also banned from names AND copy: "second brain" — Tiago Forte's BASB trademark is actively enforced, and the phrase is a cliché besides.

## Your public name: Stockpot — and why it beat Preserves

Both screened clean. Stockpot wins on what the system *is*:

1. **A stockpot transforms; preserves only keep.** Jars on a shelf are archival — sealed, static, retrieved unchanged. A stockpot simmers: things go in, break down, combine, and become something richer. That's literally the architecture — inbox → atomic breakdown → linking → wiki synthesis → deepening layers. The overnight jobs ARE the simmer.
2. **Stock is what gravy is made from.** The memory system feeds the studio's research and builds — it is the base everything else is made on. "Stockpot, by Synapsin Gravy Workshop" isn't a pun; it's an org chart.
3. **The logo already works** — the steam-becomes-mind-map mark (below) was designed before the name was chosen and happens to be a stockpot.
4. **Both words still get used:** name the encrypted backup/export layer **"preserves"** — memory put up in jars to keep. ("run preserves" as the backup command is exactly the studio's register.)

Only known collision: an iOS recipe-manager app called "Stockpot" — different category, low risk. ("Marrow" was considered and dropped: the Indian medical-ed app Marrow litigates its mark. "Commonplace" was the clean third option if Stockpot ever needs a fallback.) `stockpot.com` is taken; none needed — the public page is `gravyworkshop.com/stockpot`.

## Identity for all copy

What it is, honestly stated: a permanent memory system for one life. The memory layer is solid and in daily use (~19k atomic notes); the autonomous overnight research/build loop exists but hasn't earned its keep yet — say so, the honesty is the brand.

**Tagline (the owner's own, leads everything):**
> *stockpot: where your ideas simmer and separate.*

(It earns the word "separate" twice over — clarifying stock by skimming, and the system literally separating captures into atomic claims.)

**Site card blurbs:**
1. *a permanent memory for one life: journals and reading go in, a self-tending personal wikipedia comes out.*
2. *notes go in the inbox; overnight they become atomic claims, links, and wiki pages. it remembers so we don't have to.*
3. *a personal wikipedia that tends itself while we sleep. the memory works; the part that does research overnight is still learning.*

Longer:
> a permanent memory system for one life. journal entries, articles, and conversations land in an inbox; overnight jobs break them into atomic notes, link what belongs together, and grow a personal wikipedia that gets deeper the more goes in. the memory layer is solid and in daily use — the part that's meant to run research and builds while we sleep hasn't earned its keep yet.
