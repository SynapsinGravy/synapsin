# Identity: **Butcherblock** — short-term recall (sister to Stockpot)

Working name locked 2026-06-13. The short-term, fast half of the memory pair; **Stockpot** is the long-term half.

## What it is
A session-recall tool for people who live in the terminal (Claude Code & similar). On every compact / end-of-session hook it:
1. saves the **whole transcript** to a folder (grep-able for exact strings), and
2. **chunks the session by turn** and embeds each chunk **with no overlap**, enabling semantic search of any past moment — and a semantic hit can point grep at an exact phrase to grab just what's needed.

Sessions worth keeping get **summarized and promoted into Stockpot** (long-term). It can also run as a **local MCP** that eases/encourages searching the long-term Stockpot memory. Designed as a pair with Stockpot — either works alone, together they're a whole memory.
